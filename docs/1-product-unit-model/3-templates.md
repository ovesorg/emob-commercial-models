## Templates

### Product-Unit Definition Template

Use this template to define any Product-Unit:

```markdown
# Product-Unit: [Name]

## 1. Identity
- **Category**: [Physical / Service / Deposit-Based / Digital / Regulatory / Composite]
- **Unique ID**: [Code]
- **Description**: [Brief description]

## 2. Value
- **Customer Value**: [What customer gets]
- **Business Value**: [Revenue / retention / strategic benefit]
- **Risk Exposure**: [Liability / commitment]
- **Lifecycle Contribution**: [Revenue model / cost structure]

## 3. Context
- **Prerequisites**: [Required Product-Units or conditions]
- **Dependencies**: [Systems / assets / services required]
- **Constraints**: [Limitations / boundaries]

## 4. Obligations
- **Service Level**: [Response time / availability]
- **Warranty Terms**: [Coverage / duration]
- **Support Requirements**: [What company must provide]
- **Maintenance**: [Ongoing duties]

## 5. Costs
- **Capital Cost**: [Initial investment]
- **Operating Cost**: [Recurring expenses]
- **Logistics**: [Deployment / delivery]
- **Support Cost**: [Customer service / maintenance]
- **Depreciation**: [Asset lifecycle cost]
```

### Example: Battery Swap Subscription

```markdown
# Product-Unit: Battery Swap Subscription (Monthly)

## 1. Identity
- **Category**: Service
- **Unique ID**: SVC-SWAP-M01
- **Description**: Monthly subscription granting unlimited battery swaps at OVES swap stations

## 2. Value
- **Customer Value**: Unlimited energy access, no range anxiety, no charging wait time
- **Business Value**: Recurring revenue, customer lock-in, asset utilization optimization
- **Risk Exposure**: Asset damage, station availability commitment
- **Lifecycle Contribution**: Monthly recurring revenue minus operating cost per swap

## 3. Context
- **Prerequisites**: Active customer account, valid payment method, deposit-based entitlement
- **Dependencies**: Swap station network, battery inventory, app access
- **Constraints**: Geographic coverage area, station operating hours

## 4. Obligations
- **Service Level**: 95% station uptime, <5 min swap time
- **Warranty Terms**: Battery performance guarantee
- **Support Requirements**: 24/7 app support, station assistance
- **Maintenance**: Battery health monitoring, station stocking

## 5. Costs
- **Capital Cost**: Battery inventory allocation
- **Operating Cost**: Station operation, staff, utilities
- **Logistics**: Battery transport, balancing inventory
- **Support Cost**: App support, field assistance
- **Depreciation**: Battery cycle degradation
```
